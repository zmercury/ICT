### Switch Case statement 
Switch case statement test ant expression and execites the statement ot case equivalent to the expression. If none f the case meets the expression then default block gets executed.

Syntax: 
``` c
switch (expression) {
    case 1: {
        statement 1;
    }
    case 2: {
        statement 2;
    }
    case n: {
        statement n;
    }
    default: {
        statement;
    }
}
```
